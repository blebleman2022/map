二、快速开始（3步）
========================================

【第1步】获取高德地图 API Key（必须）

1. 访问：https://lbs.amap.com/
2. 注册并登录
3. 创建应用，获取两个 Key：
   - Web 服务 API Key（后端用）
   - Web 端 JS API Key（前端用）

【第2步】配置环境变量

编辑以下文件：

backend/.env
-----------
AMAP_API_KEY=你的Web服务API_Key

frontend/.env.local
------------------
NEXT_PUBLIC_AMAP_KEY=你的Web端JS_API_Key

【第3步】启动服务

方式一：双击 start.bat（推荐）

方式二：手动启动
  后端：
    cd backend
    python -m venv venv
    venv\Scripts\activate
    pip install -r requirements.txt
    python -m app.main

  前端：
    cd frontend
    npm install
    npm run dev

三、访问应用
========================================

前端应用：http://localhost:3000
后端 API：http://localhost:8000
API 文档：http://localhost:8000/docs

五、常见问题
========================================

Q1: 没有 LLM API Key 可以用吗？
A: 可以！不配置 LLM，系统会使用规则引擎。

Q2: 地图不显示？
A: 检查 frontend/.env.local 中的 NEXT_PUBLIC_AMAP_KEY

Q3: 搜索没结果？
A: 检查 backend/.env 中的 AMAP_API_KEY

Q4: 端口被占用？
A: 修改 backend/.env 中的 PORT=8001
